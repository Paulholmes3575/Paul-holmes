****** install notes ******
GM Castle, both closed and open variants. 
****************************************
Next to the  file name will be a comment,
New Entry or Replace Entry.
For New Entry - Place anywhere in the file (I prefer near the top).
For Replace Entry - Remove old code then place mine in.
For New File - Create a new JSON file.
****************************************

Place this file into your custom folder and add the full file name to your cfggameplay.json file. 

"objectSpawnersArr": ["./custom/closegm.json"]

If you have other files listed then just add this to the list, example 

"objectSpawnersArr": ["./custom/opengm.json","./custom/closegm.json"]

P.S, if you rename the file in the custom folder, make sure to rename it in cfggameplay.json too.

****************************************
